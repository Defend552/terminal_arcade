#Terminal Arcade

## Introduction
A terminal-based arcade game collection built in Go using the Bubble Tea TUI framework.
['Bubble Tea'](https://github.com/charmbracelet/bubbletea)

The initial game is Pong, complete with menus, difficulty selection, and game state display.

This project is a playground for learning Go, building terminal UIs, and applying state management in Bubble Tea.

---

## Features
- Terminal UI game powered by Go  
- Modular menu and difficulty system  
- Classic Pong gameplay  
- Side panel showing game state  

---

##  Demo
This is a terminal application — once built and run, you can interactively play Pong with keyboard controls.
